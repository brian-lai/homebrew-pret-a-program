# Homebrew Tap for Pret-a-Program

Homebrew formulae for [Pret-a-Program](https://github.com/brian-lai/PARA-Programming) â€” structured AI-assisted development workflow.

## Installation

```bash
brew install brian-lai/pret-a-program
```

Or explicitly:

```bash
brew tap brian-lai/pret-a-program
brew install pret-a-program
```

## Usage

```bash
pret init                      # Initialize project
pret plan add-auth             # Create a plan
pret execute                   # Create branch and track work
pret summarize                 # Generate summary
pret archive                   # Archive and reset
pret install-skills --claude   # Install Claude Code skills
```

## Updating

```bash
brew upgrade pret-a-program
```
